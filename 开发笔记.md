## 四、核心知识点清单（复习用）

- [x] 全栈分三层：**前端页面 → 后端 API → 外部 AI 服务**
- [x] Next.js 客户端组件（`'use client'`）vs 服务端组件
- [x] React `useState` 状态驱动界面 / `useRef` 跨步骤存数据
- [x] HTTP 文件上传用 `FormData`，文本传输用 `JSON`
- [x] 环境变量（`.env.local`）保护密钥，不暴露给浏览器
- [x] LLM 流式输出：`stream:true` → `ReadableStream` → `getReader()` → 逐字追加
- [x] Prompt 工程：系统提示词决定 AI 输出质量；自由格式 vs 结构化格式
- [x] 多步 AI 链：不同步骤用不同 Prompt（发散 vs 聚焦）
- [x] Prisma ORM：模型 → 迁移 → CRUD → Prisma v7 需要显式 adapter
- [x] Next.js 路由：文件即路由（`app/xxx/page.tsx` → `/xxx`），`[id]` 动态参数
- [x] 4 种文件格式自动识别（.doc/.docx/.md/.txt）

## 五、数据库模型

```prisma
// 实验报告 + AI 方向结果
model ExperimentReport {
  id        Int      @id @default(autoincrement())
  fileName  String
  rawText   String   // 解析出的报告正文
  aiResult  String   // AI 生成的方向列表
  createdAt DateTime @default(now())
}

// 选定方向 → 详细规划 + 路线图
model ProjectPlan {
  id        Int      @id @default(autoincrement())
  direction String   // 用户选中的方向
  planText  String   // AI 生成的完整方案 + 路线图
  createdAt DateTime @default(now())
}

// 简历 + JD → 定制简历
model TailoredResume {
  id        Int      @id @default(autoincrement())
  resume    String   // 用户输入的原始简历
  jd        String   // 岗位 JD
  result    String   // AI 输出的优化简历 + 调整说明
  createdAt DateTime @default(now())
}
```

## 六、API 一览

## 九、环境备忘（踩过的坑）

- **Node 版本**：必须 v22（fnm 已设默认）。`node --version` 应显示 v22.x。
- **npm 镜像**：已配 `https://registry.npmmirror.com`。
- **端口**：job-copilot 用 3100；PrismaAI 用 3000/5173。
- **Prisma v7 适配器**：schema 不能写 `url`（必须在 `prisma.config.ts`），运行时需 `PrismaLibSql` adapter。
- **`.env` vs `.env.local`**：Prisma CLI 读 `.env`，Next.js 读 `.env.local`，路径要一致。
- **DeepSeek key**：存在 `.env.local`，改了需重启 dev server。
