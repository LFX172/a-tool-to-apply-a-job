# Git 常用命令备忘

> 在 Git Bash 中执行

## 日常提交（每次改完代码跑一遍）

```bash
cd "D:/求职工作辅助工具/job-copilot"
git add -A                          # 打包所有改动
git commit -m "写清楚这次改了什么"     # 存档 + 备注
git push                            # 同步到 GitHub
```

## 查看状态

```bash
git status          # 看看改了什么文件
git log --oneline   # 查看提交历史
```
